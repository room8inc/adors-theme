# Contact Form 7 セットアップガイド

## ファイル構成

- `CONTACT_FORM7_FORM.txt` - フォームコード
- `CONTACT_FORM7_EMAIL_SENDER.txt` - 送信者宛メールテンプレート
- `CONTACT_FORM7_EMAIL_ADMIN.txt` - 管理者宛メールテンプレート
- `CONTACT_FORM7_CSS.css` - スタイリング用CSS

## ショートコード

```
[contact-form-7 id="275eb36" title="お問い合わせ"]
```

※`page-contact.php`に既に埋め込まれています。

## セットアップ手順

### 1. フォームの作成

1. WordPress管理画面にログイン
2. 「Contact」→「連絡フォーム」→「新規追加」
3. **フォーム名（タイトル）**: `お問い合わせ`
4. 「フォーム」タブを開く
5. `CONTACT_FORM7_FORM.txt`の内容をコピー&ペースト

### 2. メール設定

#### 送信者宛メール

1. 「メール」タブを開く
2. 「送信者宛メール」セクションで以下を設定：
   - **宛先**: `[your-email]`
   - **件名**: `お問い合わせありがとうございます`
   - **メール本文**: `CONTACT_FORM7_EMAIL_SENDER.txt`の内容をコピー&ペースト

#### 管理者宛メール

1. 同じ「メール」タブ内の「管理者宛メール」セクションで以下を設定：
   - **宛先**: 管理者のメールアドレス（例: `admin@example.com`）
   - **件名**: `【ADOR'S】お問い合わせがありました`
   - **メール本文**: `CONTACT_FORM7_EMAIL_ADMIN.txt`の内容をコピー&ペースト
   - **追加ヘッダー**: 
     ```
     Reply-To: [your-email]
     ```

### 3. メッセージ設定

「メッセージ」タブで以下を設定：

- **送信成功メッセージ**: 
  ```
  お問い合わせありがとうございます。内容を確認次第、担当者よりご連絡させていただきます。
  ```

- **送信エラーメッセージ**: 
  ```
  送信中にエラーが発生しました。しばらく時間をおいて再度お試しください。
  ```

- **入力エラーメッセージ**: 
  ```
  入力内容に誤りがあります。必須項目をご確認ください。
  ```

- **検証エラーメッセージ**: 
  ```
  入力内容を確認してください。
  ```

- **スパムメッセージ**: 
  ```
  入力内容がスパムとして検出されました。
  ```

### 4. CSSの追加

`CONTACT_FORM7_CSS.css`の内容を`assets/css/style.css`に追加し、`npm run build`を実行してください。

**注意**: `assets/css/style.css`に追加したCSSは、Viteでビルドされて`dist/assets/css/style.css`に含まれます。`functions.php`では`dist/assets/css/style.css`が読み込まれる設定になっています。

※既に`assets/css/style.css`に追加済みです。

### 5. その他の設定

「追加設定」タブで以下を確認：

- `mode: demo` が含まれている場合は削除（本番環境用）
