# ADOR'S (アドーズ) - 消防法・危険物法コンサルティング Webサイト構築プロジェクト

## プロジェクト概要

消防法および危険物法に関するコンサルティング事業「ADOR'S（アドーズ）」のコーポレートサイト構築プロジェクトです。

クライアントから提供された事業計画に基づき、企業の信頼性、専門性、およびコストメリットを訴求するWebサイトをWordPressで構築します。

## 戦略コンセプト

**「法令の『壁』を、利益を生む『強み』へ。」**

単なる法令順守（守り）にとどまらず、専門家によるコストカットや設計自由度の向上といった「顧客メリット（攻め）」を前面に打ち出し、選ばれる理由を明確にします。

## 目的・ゴール

* **信頼性の獲得:** 消防法・危険物という「安全・安心」に関わる分野において、高度な専門知識を有していることを伝える。

* **顧客課題の解決:** 法令遵守（コンプライアンス）、コストカット、リスク回避といった顧客のメリットを明確に提示する。

* **お問い合わせの獲得:** 事業主、建設会社、設計事務所からの相談・見積もり依頼への導線を確保する。

## ターゲットオーディエンス

事業計画書に基づく主要ターゲット:

1.  **事業主（工場・倉庫など）:** 既存・新築建物の法令適合性、防災担当者の不在・交代による不安を持つ層。

2.  **建設施工会社:** 消防設備についての専門的なコンサルティング、コストカットを求める層。

3.  **設計事務所（建築士）:** 設計段階での消防設備選定のサポート、法適合性の担保を求める層。

## デザイン・UI/UX コンセプト

* **テーマカラー:** `#fd780f` (オレンジ)

    * ロゴに使用されているカラー。炎（消防）を連想させつつ、解決への情熱やスピード感を表現。

* **デザインの方向性:**

    * **信頼感 x 躍動感:** 法令を扱う「堅実さ」と、迅速な対応力を示す「動き」のあるデザイン。

    * **参考サイト:** [株式会社ハリマニックス](https://www.harimanics.co.jp/)

        * 構成や動き（インタラクション）、情報の整理の仕方を参考にする。

        * ※完全なコピーではなく、あくまで「構成や動きの参考」とし、ADOR'Sのブランドカラーに合わせて構築する。

## サイトマップ・コンテンツ構成 (詳細)

1.  **HOME (トップページ):**

    * ターゲット別ナビゲーション（事業主様／建設会社様／設計士様）

    * 3つの強み（コスト最適化・スピード・専門性）

    * 新着情報・解決事例ピックアップ

2.  **CONCEPT (アドーズについて):**

    * 選ばれる理由、他社（メーカー等）との違い。

3.  **SERVICE (事業内容):**

    * **消防法コンサルティング:** 新築・既存診断、消防協議代行、消防計画作成。

    * **危険物施設コンサルティング:** 製造所・貯蔵所の申請支援、消火設備選定（化学工場・ガソリンスタンド等）。

    * **スマートメンテナンス:** 日常点検アプリ「スマメ」の紹介。

4.  **WORKS / COLUMN (実績・コラム):**

    * 解決事例（コスト削減実例）、法改正ニュース、お役立ちコラム。

5.  **COMPANY (会社概要):**

    * 会社情報、名古屋事業所（2025年7月開設予定）情報、対応エリア。

6.  **CONTACT (お問い合わせ):**

    * 概算見積もり依頼、スポット相談、顧問契約の相談フォーム。
