# ADOR'S (株式会社アドーズ) - 消防法・危険物法コンサルティング Webサイト構築プロジェクト

## プロジェクト概要

消防法および危険物法に関するコンサルティング事業「株式会社アドーズ」のコーポレートサイト構築プロジェクトです。

クライアントから提供された資料に基づき、「第三者的な立場で、お客様の利益だけを考えて提案する専門家（セカンドオピニオン）」というポジショニングを明確に伝えるWebサイトをWordPressで構築します。

## 戦略コンセプト

**「プロの視点によるサポート（セカンドオピニオン）」**

売り手側の事情を挟まず、お客様の立場に立った提案を徹底する。取扱製品の制約なく、本当に必要な設備だけを提案し、不透明な見積もりを一掃して納得の価格へ導く。
さらに、仕様規定に縛られない自由設計と地域ごとの指導基準への柔軟対応により、安全性とコストの最適化を同時に実現する。

## 目的・ゴール

* **中立性の明確化:** 消防設備業者でも一般コンサルでもない、独立した第三者としての立場を伝える。

* **顧客メリットの可視化:** 過剰な設備の排除、不透明な見積もりの是正、自由設計による柔軟な対応を具体的に示す。

* **問い合わせの増加:** 初回無料ヒアリングを明確にし、相談ハードルを下げる。

## ターゲットオーディエンス

主要ターゲット:

1.  **事業主（工場・倉庫など）:** 法令遵守とコスト最適化を両立したい層。

2.  **建設施工会社:** 設備更新・行政対応での専門的な助言を求める層。

3.  **設計事務所（建築士）:** 仕様規定に縛られない自由設計の支援を求める層。

## サイトマップ・コンテンツ構成 (詳細)

1.  **HOME (トップページ):**

    * ターゲット別ナビゲーション（事業主様／建設会社様／設計士様）

    * 3つの強み（中立性・透明性・柔軟性）

    * 新着情報・解決事例ピックアップ

2.  **CONCEPT (アドーズについて):**

    * 選ばれる理由、他社（メーカー等）との違い。

3.  **SERVICE (事業内容):**

    * **年間契約（継続支援）:** 月額10万円〜。定期法令チェック、行政相談・指摘対応支援、点検・記録様式整備、社内教育支援。

    * **スポット契約（個別案件）:** 案件別5万円〜。現地調査・リスク診断、是正計画作成支援、行政協議同席支援。

    * **補足:** 規模・施設数・対応範囲で調整。交通費・出張費は実費精算。まずは無料ヒアリングから。

4.  **WORKS / COLUMN (実績・コラム):**

    * 解決事例（コスト削減実例）、法改正ニュース、お役立ちコラム。

5.  **COMPANY (会社概要):**

    * 会社情報（株式会社アドーズ／代表者／所在地／事業内容）
    * 専門資格者がチームで対応する体制

6.  **CONTACT (お問い合わせ):**

    * 初回無料ヒアリングの明示
    * 相談例の具体化
    * 問い合わせ後の流れの可視化
